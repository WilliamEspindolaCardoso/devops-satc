# devops-satc

-- DevOps
DevOps é uma cultura e conjunto de práticas que unem Desenvolvimento (Dev) e Operações (Ops).
É a prática de automatizar e integrar processos para entregar software de forma contínua e eficiente.
Objetivo: Entregar software mais rápido, com mais qualidade e menos erros

-- SRE (Site Reliability Engineering)
SRE é uma abordagem criada pelo Google para garantir que sistemas sejam confiáveis, escaláveis e estáveis.
Ele aplica princípios de engenharia de software à infraestrutura e operações.
Objetivo: Garantir alta disponibilidade e desempenho dos sistemas, funcionando com confiabilidade.
